# 07 - Subnet Calculator

Takes an IP address in CIDR notation and prints the network address, broadcast address, subnet mask, host range, and total usable hosts.

## How to Run

```bash
python subnetcalc.py <cidr>
```

Example:

```bash
python subnetcalc.py 192.168.1.0/24
```
