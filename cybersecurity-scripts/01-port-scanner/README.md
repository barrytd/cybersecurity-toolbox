# 01 - Port Scanner

A TCP port scanner that checks which ports are open on a given target host.

## How to Run

```bash
python scanner.py <target> [--start-port 1] [--end-port 1024]
```

Example:

```bash
python scanner.py scanme.nmap.org --start-port 20 --end-port 100
```
