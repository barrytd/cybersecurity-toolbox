# 02 - Hash Cracker

A dictionary-based password hash cracker. Given a hash and a wordlist, it hashes each candidate word and checks for a match.

## How to Run

```bash
python cracker.py --hash <hash> --wordlist <path> [--algo md5|sha1|sha256]
```

Example:

```bash
python cracker.py --hash 5f4dcc3b5aa765d61d8327deb882cf99 --wordlist rockyou.txt --algo md5
```
