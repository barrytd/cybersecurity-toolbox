# 04 - Web Directory Enumerator

Discovers hidden directories and files on a web server by trying paths from a wordlist and reporting which ones return a non-404 status.

## How to Run

```bash
python enum.py --url <target> --wordlist <path> [--threads 10]
```

Example:

```bash
python enum.py --url http://example.com --wordlist common.txt --threads 20
```
