# Bash Scripts

A small set of shell scripts for system recon, log triage, and automation. These pair well with the Python tools above when you want a quick one-liner instead of a full program.

| Script | Purpose |
|--------|---------|
| `sysinfo.sh` | Print a snapshot of host info: OS, kernel, uptime, users, disk, memory |
| `log-analyzer.sh` | Quick stats from a log file (top IPs, error counts, etc.) |
| `backup.sh` | Compress and timestamp a directory into a backup archive |
| `user-audit.sh` | List local users, sudo group members, and recent logins |
| `ping-sweep.sh` | Ping every host in a /24 to find live machines |

## How to Run

Make a script executable, then run it:

```bash
chmod +x sysinfo.sh
./sysinfo.sh
```
